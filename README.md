  # Tic-Tac-Toe

  The classic 3×3 game for two players, in a single HTML file. Player 1 is **O**,
  Player 2 is **X**; players alternate taps on the grid.

  ## Features

  - Two-player hot-seat play on a 3×3 board
  - Win detection across all 8 lines (rows, columns, diagonals); the winning
    squares highlight in red
  - Draw detection when the board fills with no winner
  - Heading shows whose turn it is and announces the result
  - **Start** to begin and **Reset** to clear the board and counter
  - Pure vanilla HTML/CSS/JavaScript — no libraries, no backend

  ## Usage

  ```bash
  # Open in any browser
  "TIC TAC TOE.html"
  ```
  ## Tech

  Plain HTML/CSS with a dark theme and a flexbox-centered grid; game logic in
  vanilla JavaScript (verify() checks the win combinations).
